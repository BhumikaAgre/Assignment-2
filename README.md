Task 1: Check if a Number is Even or Odd
Description
This program takes an integer input from the user and checks whether the number is even or odd using an if-else statement.

Program Logic
Take a number from the user
Use modulus operator (%) to check remainder
If remainder is 0, number is even
Otherwise, number is odd

Task 2: Sum of Integers from 1 to 50 Using a Loop
Description
This program uses a for loop to calculate the sum of all integers from 1 to 50.

Program Logic

Use a for loop from 1 to 50
Add each number to the sum
Display the final result
